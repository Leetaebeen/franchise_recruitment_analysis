# 선문PC방 가맹점 모집 웹사이트

데이터 기반 PC방 창업을 위한 종합 가맹점 모집 플랫폼_V1

## 🎯 프로젝트 소개

선문PC방 가맹점 모집 웹사이트는 빅데이터 분석을 통해 예비 창업자에게 최적의 입지 정보와 데이터 기반 인사이트를 제공하는 혁신적인 프랜차이즈 플랫폼입니다.

### 주요 기능

- **📊 실시간 데이터 분석**: CSV 파일 업로드를 통한 즉각적인 상권 분석
- **🗺️ 카카오맵 연동**: AI가 추천하는 최적 입지를 지도에서 시각화
- **📈 7가지 핵심 분석 차트**:
  - 월별 매출 추이 (5-8월)
  - 재방문율 코호트 분석
  - 지역별 경쟁력 레이더 차트
  - 지역별 예상 월 매출 TOP 10
  - 연령대별 고객 분석
  - 이용시간 vs 결제금액 상관관계
- **💼 1:1 맞춤 상담**: 전담 컨설턴트와의 직접 상담 신청
- **📋 가맹 절차 안내**: 6단계 타임라인 방식의 명확한 절차 가이드

## 🛠️ 기술 스택

### Frontend
- **Framework**: Next.js 16.0.3 (App Router + Turbopack)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Charts**: Recharts
- **Map**: Kakao Maps API

### 주요 라이브러리
- `papaparse`: CSV 파일 파싱
- `lucide-react`: 아이콘
- `@vercel/analytics`: 사용자 분석

## 🚀 시작하기

### 1. 설치

```bash
# 의존성 설치
npm install
# 또는
pnpm install
```

### 2. 환경 변수 설정

`.env.local` 파일을 생성하고 Kakao Maps API 키를 추가하세요:

```env
NEXT_PUBLIC_KAKAO_MAP_KEY=your_kakao_map_api_key
```

### 3. 개발 서버 실행

```bash
npm run dev
# 또는
pnpm dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 📊 데이터 분석 방법론

### 예상 월 매출 계산

```typescript
예상 월 매출 = 평균 결제액 × max(표본 수, 50) × 30일
```

- **근거**: 일별 평균 고객 수를 표본 데이터 기반으로 추정
- **최소 보정**: 표본이 50명 미만일 경우 50명으로 보정 (신뢰도 확보)
- **월 환산**: 30일 기준으로 월 매출 산출

### 재방문율 분석

- **6월, 7월, 8월 재방문율**: 각 월의 재방문여부 데이터 집계
- **90일 재방문율**: 장기 고객 충성도 측정
- **코호트 분석**: 시간 경과에 따른 재방문 패턴 추적

### 지역별 경쟁력 지표

1. **매출 점수**: 지역별 예상 월 매출 (백만원 단위)
2. **이용시간**: 평균 이용시간 (10분 단위)
3. **재방문율**: 90일 기준 재방문율 (%)

## 🎨 디자인 특징

- **Gradient 테마**: Blue-Purple-Pink 그라데이션 일관성
- **Glassmorphism**: 반투명 배경 + 백드롭 블러 효과
- **반응형 디자인**: Mobile-First 접근방식

- **차트 설명**: 각 분석 차트 하단에 비전문가를 위한 상세 설명 추가

## 📈 데이터 무결성 원칙

> **중요**: 모든 분석 결과는 실제 CSV 데이터에 기반합니다.

1. ✅ **월별 매출 추이**: 실제 5-8월 결제 및 재방문 데이터
2. ✅ **재방문율 코호트**: 실제 6-8월, 90일 재방문 데이터
3. ✅ **지역별 분석**: 실제 지역-연령별 집계 데이터
4. ✅ **연령대별 분석**: 실제 연령대별 이용 패턴 데이터
5. ✅ **상관관계**: 실제 이용시간-결제금액 데이터

**가짜 데이터 사용 금지**: 어떤 경우에도 임의 생성 데이터를 사용하지 않습니다.

## 🗺️ Kakao Maps API 연동

### 설정 방법

1. [Kakao Developers](https://developers.kakao.com/)에서 애플리케이션 생성
2. JavaScript 키 발급
3. 플랫폼 설정에서 `http://localhost:3000` 추가
4. **카카오맵 사용 설정** 활성화 (필수!)

### 주요 기능

- **키워드 검색**: Places API를 사용한 장소 검색
- **마커 표시**: 추천 입지 위치 시각화
- **인포윈도우**: 클릭 시 상세 정보 표시
- **자동 범위 조정**: 모든 마커가 보이도록 자동 줌

## 🚧 주의사항

### CSV 데이터 형식

분석을 위한 CSV 파일은 다음 컬럼을 포함해야 합니다:

```
사용자_ID, 지역_도시, 연령대, 나이, 방문일수,
총_이용시간(분), 평균_이용시간(분), 5월_총결제금액,
6월_재방문여부, 7월_재방문여부, 8월_재방문여부, 90일_재방문여부
```

### 성능 최적화

- **표본 필터링**: 5명 이상의 표본이 있는 지역만 분석
- **데이터 슬라이싱**: 산점도는 최대 150개 데이터 포인트
- **메모이제이션**: `useMemo`를 통한 불필요한 재계산 방지
